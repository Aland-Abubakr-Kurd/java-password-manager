# • Java Password Manager

A simple console-based password manager built using Java.  
This project simulates a basic authentication system with username and password verification, password strength checking, and password generation.

---

## • Features

- User login system (username & password)
- Password verification using hashing
- Password strength checker
- Option to update password
- Random password generator
- Input validation and error handling

## • Important Note

This project is for **educational purposes only**.

- Passwords are stored using Java's `hashCode()` method  
- This is **NOT secure** and should NOT be used in real-world applications  
- Real systems use strong hashing algorithms like bcrypt or SHA-256
